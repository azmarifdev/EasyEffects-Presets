# 🎧 AZM Audio Presets

High-quality **Easy Effects presets** for clean voice recording and immersive audio experience.

Created by **AZM (azmarifdev)** for creators, developers, and Linux users who want better sound without complicated setup.

---

## ✨ Features

### 🎙️ Podcast (Voice Preset)

* Clean, noise-free microphone output
* Crisp and natural voice tone
* Optimized for **podcasting, streaming, meetings, and recording**
* Works especially well with **Fifine microphones**

---

### 🎧 Cinematic Audio (Media Preset)

* Deep bass and enhanced clarity
* Wide stereo and immersive listening
* Perfect for **music, movies, and YouTube**

---

## 📦 Presets Included

```bash
voice/
  └── azmarifdev_podcast_v1.json

audio/
  └── azmarifdev_cinematic_audio_v1.json
```

---

## ⚙️ Requirements

* Linux (PipeWire recommended)
* Easy Effects installed

---

## 🚀 How to Use

1. Open **Easy Effects**
2. Go to **Presets**
3. Click **Import**
4. Select any `.json` file from this repo
5. Apply and enjoy 🎧

---

## 🧠 Use Cases

* 🎙️ Podcast recording
* 🎥 Streaming (OBS)
* 💬 Online meetings (Discord, Zoom)

* 💻 Daily Linux usage
