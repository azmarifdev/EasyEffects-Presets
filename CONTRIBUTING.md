# 🤝 Contributing

Thank you for your interest in contributing!

## 💡 How to Contribute

- Improve existing presets
- Add new presets (voice, music, gaming, etc.)
- Fix issues or optimize sound quality
- Suggest improvements

## 📦 Guidelines

- Keep presets clean and well-balanced
- Use clear and consistent naming
- Test before submitting
- Describe what your preset does

## 🚀 Steps

1. Fork this repository
2. Add your preset in the correct folder (voice/audio)
3. Commit your changes
4. Create a Pull Request

## 📌 Note

Please keep contributions useful for real-world usage (streaming, podcast, music, etc.)
